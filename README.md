# AYANDA | Systems & Front-End Development Suite

A bespoke, high-end professional portfolio built with an editorial aesthetic, bridging clean data analytics infrastructure with refined visual intelligence. This workspace showcases interactive case studies, engineering processes, and user experience architectural frameworks.

## 🌟 Key Highlights & Architecture

*   **Refined Editorial UI:** Built using a custom typography hierarchy utilizing *Italiana*, *Cormorant Garamond*, and *Plus Jakarta Sans* for a premium, high-end editorial feel.
*   **Data-Driven Focus:** Engineered to spotlight technical analytical capabilities alongside modern front-end execution.
*   **Fluid Interactions:** Features a custom, interactive pastel-pink highlight cursor engine that mirrors live user engagement across targets.
*   **Responsive Framework:** Built entirely with a custom-compiled Tailwind CSS theme optimized for seamless cross-device rendering.

## 🛠️ Technology Stack

*   **Interface Layer:** HTML5, Tailwind CSS, Lucide System Icons
*   **Typography Engine:** Google Fonts (Italiana, Cormorant Garamond, Plus Jakarta Sans)
*   **Interaction Logic:** Vanilla JavaScript (ES6+) for dynamic viewport tracking and micro-interactions
